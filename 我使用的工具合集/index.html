windows

浏览器插件

优化浏览器体验

  1. 网页浏览体验

    • Tampermonkey（✅） ： 搭配各种脚本使用，对浏览器的浏览体验只能说百利而无一害 但要主动辨别脚本是否是安全的，以及排查浏览某些网站的时候可能会被脚本的运行而破坏正常的浏览体验
    • AdGuard（✅） : 广告屏蔽插件，支持自定义规则，但配置不当的话会导致浏览器的浏览性能严重下降
    • Dark Reader（✅） : 黑暗模式插件，夜间阅读友好
    • Stylus（⚠️） : 用于自定义网页样式，推荐搭配 userstyles.org 使用 我会用在网易云的网页修改上

  2. 下载网页资源

    • SingleFile（✅） : 将网页保存为单个 HTML 文件，适合网页归档

  3. 网页翻译

    • 沉浸式翻译（✅） : 页面段落翻译、双语对照、PDF 支持（这个需要高级会员） 支持自定义大模型 API 接口
    • 沙拉查词（✅） : 一款完全开源免费的多词典支持、划词翻译，适合英语学习者 唯一缺点是目前不支持大模型 API 翻译，但在我认识的传统浏览器翻译插件里面，这个是最好用的。[Github仓库](https://github.com/crimx/ext-saladict)
    • 通义（⚠️） : 阿里出品的翻译工具，语音识别和字幕生成 支持日语

  4. 其他功能性

    • Wayback Machine（✅） : 访问网页历史快照，适合信息回溯 给 Wayback Machine 手动提交自己特别喜欢的网站，方便以后找回

桌面应用

开发给 Windows 操作系统使用的应用

浏览器

上网工具

  1. Edge（✅） : 微软出品，不挂梯子的话同步账号密码好用
  2. Chrome（✅） : 谷歌出品，和 Edge 不相上下 对我来说这是 Edge 的下替位
  3. Firefox（✅） : Mozilla 出品，开源浏览器

ide

集成式开发环境

  1. VS Code（✅） : 微软出品的开源 IDE ，神中神 配置有点繁琐
  2. Visual Studio（✅） : 微软出品的宇宙级爆炸 C/C++/C# 方面最好用的 IDE。可以说它大块，但是不能说它支持性差
  3. IDEA（✅） : Jetbrain 出品的超神 IDE ，多用于 JAVA ,Kotlin 语言内容的开发，基本可以开箱即用 Jetbrain 的 IDE 全家桶对电脑性能要求较高，但如果电脑性能好的话，体验绝对是最好的那一个
  4. Pycharm（✅） : Jetbrain 出品的在 Python 开发中很好用的 IDE 对我来说是体验最舒服的一款给 Python 的 IDE
  5. Android Studio（✅） : 谷歌出品的专为 Android 开发原生应用的 IDE 。由 IDEA 的基础之上改版而来
  6. eclipse（❌） : 界面不好看，没有 IDEA 顺手。但性能要求比 IDEA 低
  7. Dev C++（❌） : 老旧的 C/C++ IDE，多在竞赛和上课的时候使用，但开箱即用和性能要求低是它的优点
  8. 微信开发者工具（⚠️） : 如果不是为了开发微信小程序，这将会是屎中屎的 IDE 和屎中屎一样的开发体验张小龙的🐎再给我飞一遍
  9. HBuilder（⚠️） : 一款非常轻量的国内开发的 IDE ，主要用于网页前端项目的开发
  10. Cursor（✅） : 一款目前市面上最好用的以内嵌 AI Agent 为卖点的 IDE
  11. Trace（✅） : 一款由字节跳动开发的对标 Cursor 的 IDE ，目前也算是国内这方向第一的 IDE

开发网页前后端辅助工具

开发网页前后端时提供更人性化的辅助工具

  1. Postman（❌） : 一个 API 测试工具。功能上不错，但是网络原因导致的非常不稳定
  2. ApiFox（✅） : 一个由国人开发的 API 测试工具。功能上和 Postman 差不多，但胜在网络非常稳定

vscode 插件

  2. CMake（⚠️） : 用于支持基于 CMake 构建系统的 C/C++ 项目，帮助管理和编译复杂工程
  3. CMake Tools（⚠️） : 用于支持基于 CMake 构建系统的 C/C++ 项目，帮助管理和编译复杂工程
  4. Code Runner （✅） : 为 VS Code 扩展能够支持运行几乎一般人能学的语言的支持
  5. Markdown All in One（✅） : 一个为 VS Code 扩展 Markdown 支持的插件 我会用它的快捷键支持
  6. Markdown Preview Enhance（✅） : 一个为 VS Code 扩展的 Markdown 支持的插件 我会使用这个的预览和导出功能
  7. Word Count CJK（✅） : 一个帮你像 Word 那样统计你的 Markdown 文件中写了多少字符的插件
  8. Go（⚠️） : 一个 Go 官方为 VS Code 扩展支持运行 Go 语言的插件
  9. Vue (Official)（⚠️） : 一个由 Vue 开发的为 VS Code 扩展支持 Vue 语法的插件
  10. Vue（⚠️） : 一个由 Vue 开发的为 VS Code 扩展支持 Vue 语法高亮的插件
  11. open in browser（⚠️） : 为 VS Code 使用右键快速在默认浏览器打开 HTML 文件提供支持，节省一点点翻文件夹的时间
  12. Path Intellisense（✅） : 为文件的路劲补全功能提供增加支持，无论是否开发工程项目都值得装
  13. EditorConfig for VS Code（✅） : 让自己本地浏览代码时可以以自己喜欢的代码风格来展示，并确保团队合作的效率工具
  14. Code Spell Checker（✅） : 代码拼写检查的插件。伟大，无需多言！ 当有人把 true 写成了 ture 就老实了
  15. Cline Chinese（✅） : 一款由国人汉化的 Cline 插件。用于支持 VS Code 的智能代码编写的插件 支持主流大模型的 API 接入
  16. Prettier - Code formatter（✅） : 语法格式化的神器，无论是代码还是笔记的 Markdown 都好用
  17. ESLint（⚠️） : 一个用来支持静态检查 JavaScript/TypeScript/React 的插件 虽然我给的是 ⚠️ 标签，但 JavaScript 的适用领域这么广泛，应该都装一个好点
  18. Fitten Code: Faster and Better AI Assistant（✅） : 一个辅助式的输入 AI 工具 我用于代码补全来节省重复打字的，有的时候自动补全内容需要自己把控
  19. Project Manager（✅） : 一个项目管理的插件，为 VS Code 集成快速切换不同项目的插件 其实就是快速打开收藏的文件夹
  20. Python（✅） : 一个为 VS Code 提供 Python 支持的插件 应该不会有人完全不用 Python 吧，只要摸了一点点代码之后
  21. Pylance（✅） : 一个为 VS Code 增强 Python 输入体验的插件。如代码补全提示，上下文的静态检查，输入参数提示
  22. Python Debugger（✅） : 一个为 VS Code 中使用 Python 提供 debug 模式的插件
  23. Python Environments（✅） : 一个为 VS Code 提供虚拟环境管理的插件，保证每个项目的环境不会背干扰或破坏
  24. Remote - SSH（⚠️） : 一个为 VS Code 提供远程访问计算机作为开发环境支持的插件
  25. Remote - SSH: Editing Configuration Files（⚠️） : 一个为 VS Code 增强远程开发体验的插件。允许自己配置文件，自定义语法着色
  26. Remote Explorer（⚠️） : 一个用于管理 VS Code 中需要连接多个不同远程计算机的插件 和 Project Manager 一个性质
  27. Thunder Client（⚠️） : 一个用于在 VS Code 内测试后端 API 的插件 和 Postman 一样
  28. WSL（🧪） : 一个为 VS Code 提供支持连接 WSL 并支持使用 WSL 上的工具链的插件
  29. TODO Highlight（⚠️） : 一个用于提示代码哪部分 TODO 的插件 当然拿来当作自己的 TODO 类软件也不是不行，但真的会有人拿着个来当自己的主用 TODO 吗
  30. Bocchi The Rock（⚠️） : 一个好用的插件合集推荐的插件 我上面有相当一部分都是在这里知道的
  31. Genshin Impact Theme（✅） : 一款为 VS Code 开发的原神主题 VS Code 中主用这个主题
  32. One Dark Pro（✅） : 一款超高人气的暗紫色主图 我在其他地方如果能设置这个为主题的话，我首选这个作为配色方案

各种小工具

用于提升电脑使用体验的各种可折腾的小工具，没有特别指出的话默认支持中文

  1. 图吧工具箱（✅） : 装机必备工具，包含电脑上各种硬件检测的的实用工具和一些使用的软件工具
  2. LocalSend（✅） : 一款开源的局域网内传输文件的工具，支持 Android、Windows、Linux、Mac OS 多平台之间的传输 可满足日常琐碎的小文件传输需求，如果需要大文件传输，还是不要用这个了
  3. Internet Download Manager（✅） : 一款超好用的可多线程下载的下载器，美中不足的是不支持磁力链方式的下载。其他时候使用体验很好 盗版遍地跑，但有能力还是支持正版
  4. Calibre（✅） : 一款好用的电子书阅读器，主流的常用电子书格式基本都会支持。可开启本地服务器，通过浏览器访问服务器上的所有电子书内容 除了界面有点丑就没什么缺点了，但也支持自己使用 CSS 来修改软件的样式
  5. Obsidian（✅） : 一款 Markdown 笔记软件。启动速度很快、社区的第三方插件数量超多、界面好看、支持把本地的知识库部署到网页上、支持使用官方或自定义的笔记备份方案 可探索程度非常高
  6. MTool Box（✅） : 一款简易的 Windows 美化工具，支持自定义白天与黑夜的不同主题样式
  7. Mem Reduct（✅） : 一款开源的内存清理工具，当你使用运行内存不太够用的机器时，不妨来使用这个由 C 语言编写的小工具。及时清理掉在后台的无用垃圾
  8. Windows Auto Dark Mode（✅） : 一款开源的可自动/手动切换 Windows 白天/黑夜模式的软件，主要由 C# 语言开发
  9. BitComet（✅） : 一款磁力链接/种子 下载上传好用的工具，可支持的第三方工具较少 建议直接找破解版提升体验
  10. qBittorrent（✅） : 一款磁力链接/种子 下载上传好用的工具，可支持的第三方工具多
  11. PeerBanHelper（✅） : 一款设计给磁力工具的开源的免费防火墙工具 抵制 PCDN ，人人有责。维护良好的磁力环境
  12. Subtitle Edit（✅） : 一款开源免费的字幕处理工具，主要由 C# 开发
  13. BetterGI（✅） : 一款开源的设计给《原神》自动化的脚本软件 通过模拟键鼠输入+实时图像识别，不会有封号风险
  14. MoonLight（✅） : 一款开源免费的视频串流工具，只要作为服务器的一方机器性能好，服务器方和客户端方的网络情况好的情况下，该软件就是串流的最好工具。支持 Android、Windows、Linux、Mac OS 多平台
  15. Sunshin（✅） : 作为 MoonLight 官方指定的开源免费服务器端部署工具，支持 Android、Windows、Linux、Mac OS 多平台
  16. ZeroTier（✅） : 一款用于实现内网穿透的工具，支持 Android、Windows、Linux、Mac OS 等多平台
  17. PotPlayer（✅） : 一款多媒体播放器，内置了很多功能，支持外挂字幕、录制视频、截图、观看网络流视频 但需要自己手动去找配置教程才能有更好的体验，默认设置不太好，高版本也有内置广告，但可以找方法干掉
  18. VLC media player（✅） : 一款很好用的开源免费跨平台的多媒体播放器 可以去找懒人包，一步到位。但听说对 HDR 的支持不太完整
  19. 弹弹 Play（✅） : 一款免费的支持本地/网络弹幕的多媒体播放器。主要用于动画的观看。支持自动获取并刮削视频封面、开启服务器模式在其它设备观看
  20. FFmpeg（✅） : 一款开源免费的超神命令行音视频处理工具，很多音视频播放器、处理工具都是基于这个项目搭建起来的
  21. MediaCoder（✅） : 一款强大的图形化音视频处理的软件 翻译：界面化封装的 FFmpeg

  5. Watt Toolkit（✅） : 一个半公益性的免费网络代理工具，有很多在游戏方面使用的网络代理加速功能 包含 Github 加速。曾用名是：Steam ++

平面设计

处理图像的高级软件放这里了

  1. PhotoShop（✅） : 一款由 Adobe 开发的地球上我觉得最好用的图像处理工具，超级强大的图片修改和样式修改，美区支持 AI 生成式内容。软件的使用体验和自己的电脑性能呈正相关 非商用内容直接找破解版，不用思考任何风险（
  2. LightingRoom（✅） : 一款由 Adobe 开发的，从 PhotoShop 中分离出来的调色模块为单独的软件，对批处理大量图片有一定的优化
  3. Illustrator（⚠️） : 一款由 Adobe 开发的专门用于制作矢量图的软件，通常用于制作产品 Logo 和简单的、具有设计感的各种图标图案 要是硬说PS也可以做，所以这个对一般人来说仅仅做一个介绍
  4. SAI2（✅） : 一款超级轻量的绘画软件，空间占用小，性能要求低。但如果工程项目真的大起来的话，对内存要求和 PS 一样高，CPU 也是。但这个软件并没有 GPU 加速，所以如果工程大的话还是推荐实用 PhotoShop 博主画画最喜欢用这个，更顺手，如果可以请支持正版，但正版并没有中文，需要自己再去找有中文的免费盗版

版本控制工具

就是控制一个工程项目的版本的工具

  1. Git（✅） : 一款流行的免费开源版本管理工具，多用于小文件居多，且项目体积不大的项目中
  2. SVN（✅） : 一款流行的免费的版本管理工具 据了解多用于游戏行业的版本管理和美术行业的版本管理
  3. Github（✅） : 地球上最流行的在线代码托管平台 好用爱用，还有很多可以白嫖的好用功能，如 Github Action
  4. Gitee（⚠️） : 一个对 Github 拙劣模仿的国内的代码托管平台（ ，如果不是 Github 网络连接不太稳定，我相信很多个人是不会选择这个用作分发平台的

3D CG 领域

和3D模型有关的一切

  1. Blender（✅） : 一款开源免费的超级强大的 CG 软件，拥有超强大的第三方社区插件支持。一款仅仅 500MB 不到的软件，可以实现高效的模型建造、雕刻、动画制作、骨骼绑定、物理模拟、特效制作等等几乎 CG 领域需要做的所有画面工作都能胜任 如果你不会那就是你的问题（ ，社区总会有解决方案的出现，社区活跃也是它非常强大的一个原因。但是硬伤是对超多面的大场景处理应不从心
  2. MikuMikuDance（✅） : 一款主要面向技术宅的集成 3D 建模、动画、物理模拟、镜头动画的新人友好型 CG 软件 但是现在社区基本转移到了 Blender 那边去了，所以社区支持不是很好了，出问题大概率要自己找方法解决
  3. MikuMikuMoving（✅） : 一个改良版精简版的 MMD 版本 由于我没用过，这是 @NisssOwO 提出来让我加的
  4. PmxEditor（✅） : 一款专为 .pmx 格式的模型进行编辑的工具 这是 @NisssOwO 提出来让我加的
  5. 3D MAX（✅） : 一个科班队的来自 Autodesk 开发的专注于建模的 CG 软件，要钱，但商业支持好 但有人吐槽说这几年 Autodesk 吃相太难看，年年挤牙膏，订阅费还涨了不少
  6. Maya（✅） : 一款来自 Autodesk 开发的专注于动画领域的 CG 软件，Blender 可能会出问题，但 Maya 绝对不能出问题，因为它也是收费的
  7. Houdini（🧪） : 一款地表最强大的特效领域软件，非常多的工业特效电影都会采用这个软件作为解决方案，但同时也对设备性能提出了极其高的要求

音频处理及制作

音乐人吃饭用的和处理音频时会使用的

  1. FL Studio（✅） : 一款强大的音频处理工具，很多的音乐人都会使用这款软件。社区活跃，插件丰富，就是软件定价有点贵 但是对于有钱音乐人来说这点软件费用，对比他们在硬件上的支出应该只是九牛一毛罢
  2. Synthesizer V（✅） : 一款由华人在日本开发的以 AI 合成为卖点的虚拟歌姬软件。软件本身有一点点贵，但是如果找好想要用哪个声库再买捆绑销售的方式性价比就比较高 AI 的加成下让软件非常强大，但是声库稍微有点少。我是夢ノ結唱的粉丝啊！×
  3. Vocaloid（✅） : 一款由雅马哈出品的虚拟歌姬软件。特点是非常强大的声库支持，但是体验过 Synthesizer V 的人应该不会想再用这个了罢（
  4. Open Utau（✅） : 一款开源免费的虚拟歌姬软件，如果需要制作一首歌内多个歌姬的歌唱可能用起来会比较痛苦。但优点就是他的开源免费和稍微算是活跃的社区了 自从 Synthesizer V 推出之后，这个的光辉在逐渐减淡了
  5. Melodyne（⚠️） : 一款专门用于调音的软件 博主认识这个的时候还是因为想学做鬼畜来着， B站UP主 @还有一天就放假了 当时还有个课程就是介绍了这个和 Utau 我才认识的，现在传统派的调音也没有多少人做了，做的好多也是抽象向的鬼畜东西了。又是令人怀念的那一段时间，但科技的发展就是会这样
  6. Adobe Audition（✅） : 一款由 Adobe 开发的专注于音频处理的软件，据了解多用于进行音频的提取、降噪、音效合成方面的工作 但现在来说个人感觉有很多的开源工具都能快速做到除了音效合成方面都能有很不错的效果

视频剪辑

剪辑蒙太奇的魅力时刻

  1. Adobe Premier Pro（✅） : 一款由 Adobe 开发的作为生产力多人合作的没有多少替代品的剪辑软件，可以对接 Adobe 自家的其他产品 如 Adobe Effect
  2. Adobe Effect（✅） : 一款由 Adobe 开发的专为视频领域做特效合成的软件，主要搭配 Adobe Premier Pro 使用
  3. Adobe Media Encoder（⚠️） : 一款由 Adobe 开发的专注于给自家产品用于素材管理编码的软件，主要解决不同设备不同录制规格之间的兼容问题
  4. Sony Vegas Pro（⚠️） : 一款由 Sony 推出的剪辑软件。软件性能挺一般，界面也一般 我认识这个也真的是因为鬼畜调音，然后做了个垃圾（
  5. 剪映（✅） : 来自字节跳动的强力作品。有很多好用的 AI 工具和实时性很强的素材和经典 meme 都能轻松找得到 就是需要冲会员体验才好
  6. 必剪（❌） : 一款由哔哩哔哩推出的唯一真屎剪辑软件，AI 功能没有剪映好用，素材库没有剪映好。剪辑体验不如 Adobe Premier Pro ，导出性能不如 Adobe Premier Pro，最大投稿稿件大小不如自家网页端 屎到家了

ai 生成式内容

关于 AI 会接入的一些生产力工具

  1. ComfyUI（✅） : 一个以节点控制为思路的生成图片、视频、模型的超级强大的 AIGC 工具
  2. VoiceTransl（✅） : 一款开源免费的一站式离线 AI 视频字幕生成和翻译软件，从视频下载，音频提取，听写打轴，字幕翻译，视频合成，字幕总结各个环节为翻译者提供便利的工具

android

自动化小工具

释放双手，体验看监控的乐趣

  1. Smart-AutoClicker（✅） : 一款开源免费的支持图片识别和工作流创建的自动点击的强大工具
  2. 李跳跳（✅） : 一款强大的开源免费基于应用的文字识别的自动点击实现快速跳过广告的工具

多媒体工具

手机也是需要利用好这块屏幕和摄像头的

  1. DroidCam（✅） : 一个可以在电脑端中的 OBS 通过这个插件和手机上安装这个软件实现电脑可以在局域网中调用手机摄像头，实现电脑免费扩展高清摄像头的一个途径 免费版会有水印
  2. XPlayer（✅） : 一个基本可以播放所有主流视频格式的视频播放器， Youtube 风格的播放器行为控制风格
  3. Snapseed（✅） : 一款由谷歌推出的免费的图片处理软件，和 Adobe LightingRoom 的定位差不多的软件
  4. 照片编辑器（✅） : 一个免费的图片编辑器，一个简陋版本的 Adobe PhotoShop
  5. MoonLight（✅） : 一款开源免费的视频串流工具，只要作为服务器的一方机器性能好，服务器方和客户端方的网络情况好的情况下，该软件就是串流的最好工具。支持 Android、Windows、Linux、Mac OS 多平台
  6. space desk（✅） : 一款实用的投屏软件。支持使用 USB、WIFI/LAN 进行视频传输，支持 Windows、Android、Mac OS 只支持使用 Windows 作为被投屏的方式

学习工具

学习方面和效率方面的工具

  1. NotebookLM（✅） : 一款由谷歌推出的私人知识库 AI 助手，有强大的根据用户主动提供的内容思维导图生成、电台生成、要点总结等各种实用性功能
  2. WakeUp 课程表（✅） : 一款设计给大学生及以上的学生群体的课程表 APP，支持自定义课程表外观和在线分享课程表等实用功能
  3. 英汉词典（✅） : 一款可离线快速查询英语单词、认识单词意思、练习单词拼写、根据中文辨别英文单词意思选择的的功能，与介入了 AI 对话帮助学习的的软件 以前还有根据领域不同的专门使用哪个部分的单词呈现。然后那个接入的 AI 也其实没什么用，至少对我来说是这样
  4. 边走边听背单词（✅） : 一款专门用于学习记忆英语单词的软件，从小学生到研究生都适用 这个作者的其他软件其实也推荐试试
  5. 羊羊粤语（✅） : 属于广东人的新华字典
  6. Obsidian（✅） : 一款 Markdown 笔记软件。启动速度很快、社区的第三方插件数量超多、界面好看、支持把本地的知识库部署到网页上、支持使用官方或自定义的笔记备份方案 可探索程度非常高
  7. 菜鸟教程（✅） : 一个对于中国宝宝体质来说学习入门编程的绝佳地方 APP 其实只是网页套壳，所以可以直接快速启动网页就可以了（
  8. 调音器，节拍器（✅） : 一款免费的用于乐器练习、调音演奏的软件

实用小工具

用于提升手机使用体验的各种小工具

  1. 搜图神器（✅） : 集齐壁纸下载、图片识图、很多实用小工具于一体的应用 界面还挺好看的
  2. 混合盘（✅） : 一个免费的用于搜索网站规则引擎 我主要用来当作一个网盘资源搜索器
  3. LocalSend（✅） : 一款开源的局域网内传输文件的工具，支持 Android、Windows、Linux、Mac OS 多平台之间的传输 可满足日常琐碎的小文件传输需求，如果需要大文件传输，还是不要用这个了
  4. 一个木函（✅） : 一个实用的工具箱软件 在保持软件强大性的同时，界面设计的也不错
  5. 小米计算器（✅） : 一个实用强大的手机计算器 但是没找到官方的网站，只能自己去找资源了

极客与进阶工具

此部分板块需要有一定的计算机知识作为支撑

  1. Termux（🧪） : 一个终端模拟器，可以实现容器化启动 Linux 实现一些 Android 无法实现的高级功能
  2. Magisk（🧪） : 一个用于搞机的软件，获取 root、安装模块
  3. LSPosed（🧪） : 一个用于 hook 各种东西的框架
  4. scene（🧪） : 一个用于修改手机调度和查看各种硬件的工作信息的软件
  5. Mono（⚠️） : 一个用于检查自己的 Android 设备环境是否异常的软件
