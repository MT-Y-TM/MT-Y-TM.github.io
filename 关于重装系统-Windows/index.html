关于安装系统(Windows)

发表于 2025-09-05 分类于 Windows
本文字数： 867 阅读时长 ≈ 1 分钟
一篇记录我给 zzp 的 华为 Mate 14 D 2022 装 Windows ，最后面卖给〇转卖了 1800 的日记，还要祭奠我死去的一块U盘。祷管、作哀

使用工具

  1. 微 PE
  2. 一块 在并夕夕上买的 8G U 盘
  3. 一台起码能正常开机的电脑

最佳实践

本文章建立在想要在一块完全没有任何数据的硬盘上安装 Windows 系统为前提

  1. 先下载 微 PE ，往你的 U盘 里面安装好 PE 系统
  2. 去微软的官网下载 Windows系统ISO镜像 这里放的是 Windows 11 的下载地址
  3. 下载完 Windows系统ISO镜像 后，往你在 此电脑/我的电脑 中找到一个你的 U盘 的盘符，点进去，把你下载好的 ISO 复制进去
  4. 先安全的退出你的 U盘 ，再插到你想要安装系统的电脑的 USB插口 上。在开机的时候，进入 BIOS （自己查自己的主板应该如何进 BIOS ），在里面找到 允许通过USB设备启动 ，保存退出。
  5. 重启电脑时会自动进入到 PE系统 里面。启动 PE系统 自带的 DiskGenius（分区精灵）
  6. 在硬盘上右键，选择 快速分区 （这里就可以跳过一系列繁琐的配置）
  7. 再根据自己的情况决定要不要把整块硬盘作为单独的 C盘（系统盘） 或切出一些空间用来作为其他盘符
  8. 回到 PE系统 的桌面上，有一个是 Windows 安装器，启动它
  9. 在里面选择安装文件。在之前上文让你下载的 Windows系统ISO镜像 ，选择它
  10. 选择安装位置。这里是让你选择系统要安装在哪个盘符里面，按照默认来说就是直接装在 C盘 上
  11. 等待安装完成，完成后会提示重启电脑。在完全关机好后，拔出 U盘 。
  12. 此时进入到电脑后就是
