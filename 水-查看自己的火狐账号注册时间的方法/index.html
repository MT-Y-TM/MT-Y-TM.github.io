(水)查看自己的火狐账号注册时间的方法

发表于 2025-08-28 分类于 Firefox
本文字数： 758 阅读时长 ≈ 1 分钟
一个查看自己火狐账号的注册时间的方法

前言

今天在迁移火狐浏览器的数据，以为自己应该是没注册国际版的账号的。结果下好国际版登录账号的时候，我都做好要注册账号的准备了，结果点下一步居然不是提示账号需要注册，当场改了个密码就登上去了

Firefox用户界面

不经让我好奇，我的账号是什么时候注册的，结果我找了一会发现官方提供的设置里面没有。 但是！我可是学了将近三年网页开发的 MAN ，这点去区区小问题怎么能难道我这个小兵呢（ 于是我想起来以前在看B站的注册时间的那个方法，妹想到 真的可以看得到

方法

  1. 首先，我们需要进入到我们火狐浏览器的账号页面 https://accounts.firefox.com/ ； 或者你主动点一下你火狐浏览器右上角的那个头像也是可以的
  2. 对页面中的任意位置鼠标右键或按 F12，进入到开发者工具（检查），找到 网络 一栏
  3. 对页面刷新，获取到一些信息文件，在网络下面一栏的筛选条件下找到 XHR
  4. 寻找 POST 方法下的域名为 graphql.accounts.firefox.com 的文件，点击它，会跳出来一个新的页面
  5. 在跳出来的新页面点击 响应 ，你会看到一个 JSON 格式下的只有一个 0 { data: {…} } 的内容
  6. 在里面点开你会看到一个 data { account: {…}, session: {…} }，再点开一层 account: {…} 你会看到里面存折你的 id，accountCreated:, passwordCreated 这三个信息，里面的 accountCreated 就是你的注册时间了，复制一下里面的一长串数字
  7. 在网上随便找一个时间戳转换器，把单位调成毫秒，你就可以看到你的火狐账号的具体注册时间了
